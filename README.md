# ☕ CoffeeShopApp — Кофейня / Меню

Каталог кофейных напитков с поиском и карточками товаров.

## 📱 Screenshots
*(Скриншоты будут добавлены — запусти эмулятор в Android Studio)*

## 🛠 Tech Stack
- **Kotlin** + Coroutines
- **MVVM** Architecture
- **Room** Database (меню)
- **LiveData** + **ViewBinding**
- **RecyclerView** + **CardView**
- **Glide** (загрузка изображений)

## 📁 Project Structure
```
app/src/main/java/com/example/coffeeshop/
├── data/   — MenuDao, AppDatabase, MenuRepository
├── model/  — MenuItem entity (name, price, category, imageUrl)
└── ui/     — MainActivity, MenuViewModel
```
